# Hotel Booking Management System (Book My Stay App)

## Author

bhanu

## Project Overview

The **Hotel Booking Management System** is a console-based Java application developed as part of App-Based Learning.
The goal of this project is to demonstrate how **Core Java and fundamental data structures** can be used to build a real-world application that manages hotel room bookings.

Instead of focusing on UI design, the system concentrates on **data structures, object-oriented programming, and system logic** to ensure efficient and reliable booking operations.

---

## Objective

The main objective of this project is to design and implement a system that simulates how hotel booking platforms manage room reservations while maintaining:

* Fair request processing
* Accurate room availability tracking
* Prevention of duplicate or conflicting bookings
* Clean and maintainable system design

This helps students understand how theoretical concepts are applied in real software systems.

---

## Key Concepts Used

### Core Java

* Classes and Objects
* Encapsulation
* Methods and Constructors
* Exception Handling
* Collections Framework

### Data Structures

The system uses different data structures to solve specific problems:

* **Queue (FIFO)** – Handles booking requests in the order they are received.
* **ArrayList / List** – Stores available rooms and booking records.
* **HashSet / HashMap** – Ensures unique bookings and quick lookup of room information.

---

## System Features

The application demonstrates basic hotel booking operations such as:

* View available rooms
* Create a booking
* Maintain booking records
* Prevent double booking
* Process booking requests fairly using FIFO order

All operations run in a **console-based environment** for easier debugging and understanding of program logic.

---

## Learning Approach

The project is built **incrementally**, where each stage introduces a new concept while reinforcing previously learned ones.

Each stage focuses on:

* Understanding **why a data structure is used**
* Identifying **the problem it solves**
* Recognizing **limitations of simpler implementations**
* Refactoring the system to make it more scalable and maintainable

This approach reflects how real software systems evolve over time.

---

## Scope of the Project

The project intentionally focuses on:

* Data structures implementation
* Core Java programming
* Object-Oriented Design

User interface design and advanced input handling are kept minimal so that the emphasis remains on **core system behavior and logic**.
