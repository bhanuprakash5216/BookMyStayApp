
# 🏨 BookMyStayApp

A complete Hotel Booking Management System built using Core Java and Data Structures.

## 🚀 Features

### 🔍 UC4 – Room Search
- View available rooms
- Read-only inventory access

### 📥 UC5 – Booking Queue
- FIFO request handling using Queue

### ✅ UC6 – Room Allocation
- Prevents double booking using Set
- Inventory consistency maintained

### 🎁 UC7 – Add-On Services
- Multiple services per reservation
- Cost aggregation

### 📊 UC8 – Booking History
- Stores confirmed bookings
- Generates reports

### ⚠️ UC9 – Validation & Error Handling
- Custom exceptions
- Fail-fast design

### ❌ UC10 – Cancellation
- Rollback using Stack (LIFO)

### 🧵 UC11 – Concurrency
- Thread-safe booking system
- Prevents race conditions

### 💾 UC12 – Persistence
- Saves system state to file
- Restores data on restart

---

## 🧠 Concepts Used
- OOP (Encapsulation, Abstraction)
- Data Structures (Queue, Stack, List, Map, Set)
- Multithreading & Synchronization
- File Handling (Serialization)
- Exception Handling

---

## 🎯 Outcome
A scalable and robust backend system simulating real-world hotel booking operations.
